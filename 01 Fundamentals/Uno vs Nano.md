# Uno vs Nano

**Tags:** #fundamentals #hardware

---

## Comparison

| Feature | Arduino Uno | Arduino Nano |
|---|---|---|
| Size | Credit card (~69×53mm) | Stick of gum (~45×18mm) |
| Microcontroller | ATmega328P | ATmega328P (same) |
| Clock speed | 16 MHz | 16 MHz |
| Digital pins | 14 | 14 |
| Analog pins | 6 | 8 |
| Flash | 32KB | 32KB |
| SRAM | 2KB | 2KB |
| USB connector | Type-B (square) | Mini-USB |
| Best for | Learning, prototyping | Compact projects |

---

## The Key Point

They use the **same microcontroller**. All programming knowledge
transfers 100% between them. The physical size and connector
are the only meaningful practical differences.

This course used the Uno. All project code works on the Nano
without modification.

---

## When to Choose Each

**Uno:** Easier to handle on a breadboard. Larger pins.
Better for learning and debugging.

**Nano:** Fits inside finished enclosures. 2 extra analog pins.
Better for compact final builds.
